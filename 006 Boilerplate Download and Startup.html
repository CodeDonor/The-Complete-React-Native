Attached to this lecture is a zip file. Download this zip file and extract it somewhere on your machine - we will use these files in the App Setup lecture video.

Important - This boilerplate is updated frequently and requires some amount of time for Udemy to propagate the file change across the world. If for any reason you cannot download the boilerplate, please download from this Github repository.

To clone the repository, run this command in your terminal:

git clone https://github.com/0x-cygnet/rn-starter.git

Startup Notes

In the next lecture, we will be installing dependencies and starting up the application. With the latest version of Expo, the web UI that shows the QR code at localhost:19002 as seen in the video no longer exists. This web UI has been fully removed from Expo moving all of the settings including the QR code to the terminal output:

https://blog.expo.dev/sunsetting-the-web-ui-for-expo-cli-ab12936d2206

Windows users should use PowerShell to run their application (or if you are an advanced user, the WSL terminal). Git Bash currently does not work with Expo and will not display the QR Code or options.
